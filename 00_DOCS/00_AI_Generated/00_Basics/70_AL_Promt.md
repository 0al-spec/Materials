# Контекст
## Zero-Trust Agent Layer (0AL)
- Миссия: каждое приложение > агент с декларацией (паспортом) и sandbox.
- Ключевые компоненты:
  - `Agent Passport`: безопасная декларация полномочий и политик (YAML + подпись);
  - `agentifyd`: демон агентирования бинарников по паспорту;
  - `zeroald`: референсный рантайм для запуска агентов в zero-trust;
  - `Hypercode`: декларативное описание логики/поведения (файлы `.hs` и `.hcs`).
- Особенности:
  - Агент = процесс + паспорт; паспорт — кто, что может, кто подписал.
  - Все агенты — проверяемые и изолируемые; важна цепочка доверия, discoverability, enforceability, auditability.
